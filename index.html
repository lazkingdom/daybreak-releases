Install steps

  1. Tap "Download for Android" above and save the file
  2. Open the downloaded file to install it
  3. If prompted, allow installing from this source
  4. Open Daybreak and tap "Have an invite code? Join your church"
  5. Scan the QR code (or enter the code) from your church admin
